### [AI 세미나: Agentic AI와 프롬프트 설계](./ai-seminar-note.md)

- **프롬프트 설계** — COSTAR 원칙과 프롬프트 체이닝으로 요구사항을 명확히 전달하고 복잡한 작업을 단계별로 나누는 방법을 다룬다.
- **RAG** — 임베딩·지식 기반·검색을 이용해 LLM의 최신성·정확성을 보완하고 환각을 줄이는 구조를 설명.
- **MCP와 도구 통합** — 모델과 외부 서비스·데이터 소스를 표준 방식으로 연결하고 안전하게 도구를 호출하는 방법을 정리.
- **ReAct·컨텍스트 관리** — 추론과 행동을 반복하는 에이전틱 루프, 토큰 비용과 대화 맥락을 관리하는 실무 원칙을 소개.
