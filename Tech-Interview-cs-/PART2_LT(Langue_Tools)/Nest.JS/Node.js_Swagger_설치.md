# NestJS Swagger CLI 플러그인 — 한글 번역 및 구현 가이드

## 개요

TypeScript의 메타데이터 리플렉션 시스템은 몇 가지 한계가 있어, 클래스가 어떤 프로퍼티로 구성되어 있는지 파악하거나 특정 프로퍼티가 선택적인지 필수인지 판단하기 어렵다. 하지만 이런 제약 중 일부는 컴파일 타임에 해결할 수 있다.

Nest는 TypeScript 컴파일 프로세스를 강화하여 반복적인 보일러플레이트 코드를 줄여주는 플러그인을 제공한다.

> 💡 **힌트** 이 플러그인은 선택적으로 적용할 수 있다. 모든 데코레이터를 직접 선언하거나, 필요한 곳에만 특정 데코레이터를 사용하는 방식도 가능하다.
