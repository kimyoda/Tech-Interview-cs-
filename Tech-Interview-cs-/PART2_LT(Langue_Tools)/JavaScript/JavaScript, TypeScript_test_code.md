# 🌐 JavaScript/TypeScript 테스트 코드

- 테스트 코드는 프로덕션 코드가 의도대로 동작하는 지 자동으로 검증하는 별도의 코드다.
- JavaScript/Node.js에서 환경에서 테스트는 코드의 신뢰성을 높여주고, TypeScript를 사용하면 타입 안전성을 추가로 검증할 수 있다.

---

### 1. 개념

- JavaScript 활용: 동적타입으로 인해 런타임 오류가 발생할 수도 있으나, 테스트로 이를 사전에 검증한다. Jest라는 라이브러리를 주로 활용한다.
- TypeScript 활용: 정적 타입 체크와 함께 테스트로 런타임 동작을 검증한다. `@types/jest`같은 타입정의로 타입 안전성을 유지한다.
